Venture

Venture is a location-based storytelling hybrid mobile app. User can post short stories (or ventures) at their current location, leveraging device geolocation services to set that venture's position. Other users must be within a 50-foot radius of the venture in order to read that venture's content. Users can view stories that are out of range on a map and tap them to open google maps directions to that story.

Weather or Not

Weather Or Not is a mobile-first web app that first gives the user a random location. They are then tasked with guessing the current temperature at that place. Points are accumulated based on how close their guesses are to the correct temperature.

Technology

Delivery Pal is build with Ionic, Angular, and Firebase for real-time location tracking. Integrating with the Google Maps API for address lookup and navigation. A Node.js server hosted on Heroku acts as a middle-man, handling backend logic and database updates within Firebase
